1.Django არის Python-ზე დაფუძნებული ღია კოდის (open-source) ვებ-ფრეიმვორკი. მისი საშუალებით შესაძლებელია ვებ-აპლიკაციების სწრაფად და უსაფრთხოდ შექმნა.
2.Full Stack Framework – Django მართავს როგორც სერვერულ (Back-End), ისე მომხმარებლის ინტერფეისის (Front-End) ნაწილს თავისი Template სისტემის გამოყენებით.
Back-End Framework – Django მხოლოდ სერვერულ ლოგიკას და მონაცემთა ბაზას მართავს, ხოლო Front-End კეთდება ცალკე ტექნოლოგიებით, მაგალითად React, Vue ან Angular.
3.სწრაფი განვითარება
მაღალი უსაფრთხოება
ჩაშენებული ადმინისტრაციული პანელი
ORM-ის მხარდაჭერა
დიდი დოკუმენტაცია და საზოგადოება
4.Django იყენებს MVT (Model-View-Template) არქიტექტურას, რომელიც MVC-ის (Model-View-Controller) მოდიფიცირებული ვერსიაა.
5.Model – მუშაობს მონაცემთა ბაზასთან.
View – ამუშავებს მოთხოვნებს (Requests) და აბრუნებს პასუხებს (Responses).
Template – HTML ფაილები, რომლებიც მომხმარებელს ეჩვენება.
6.ORM (Object Relational Mapping) არის სისტემა, რომელიც საშუალებას გვაძლევს მონაცემთა ბაზასთან ვიმუშაოთ Python-ის ობიექტების საშუალებით SQL კოდის დაწერის გარეშე.
7.Django გამოვიყენოთ როცა:

პროექტი დიდია
საჭიროა უსაფრთხოება
საჭიროა ავტორიზაცია, ადმინისტრაცია და მონაცემთა ბაზასთან აქტიური მუშაობა
საჭიროა სწრაფი განვითარება

Light Stack გამოვიყენოთ როცა:

პროექტი პატარაა
საჭიროა მაქსიმალური წარმადობა
მხოლოდ API-ის შექმნა გვჭირდება
მინიმალური ფუნქციონალი გვაქვს
8.Django-ში Application (აპლიკაცია) არის პროექტის დამოუკიდებელი ნაწილი, რომელიც კონკრეტულ ფუნქციონალს ასრულებს.
