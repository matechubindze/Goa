node_modules

ეს არის საქაღალდე, სადაც npm ათავსებს ყველა იმ პაკეტს/ბიბლიოთეკას, რომელიც პროექტს სჭირდება.
შიგნით ათასობით ფაილიც შეიძლება იყოს — ყველაფერი, რასაც თქვენი კოდი იყენებს.

src

პროექტის "გული".
აქ ინახება თქვენი რეალური კოდი — კომპონენტები, სტილები, სერვისები, ლოგიკა და ა.შ.

app.jsx

ეს ფაილი ხშირად წარმოადგენს აპლიკაციის მთავარ React კომპონენტს.
შიგნით არის ის UI, რომელიც პირველ რიგში უნდა გამოიტანოს აპლიკაციამ.

main.jsx

ეს არის React-ის საწყისი "შემყვანი" ფაილი.
აქ ხდება app.jsx-ის შეტანა DOM-ში და აპლიკაციის ჩატვირთვა ბრაუზერში.

.gitignore

ეს ფაილი აწერს Git-ს, რომელი ფაილები არ უნდა შეინახოს.
მაგალითად: node_modules, build ფაილები და სხვა არასაჭირო რამეები.

package-lock.json

ამ ფაილში ზუსტად ინახება რომელი ვერსიები არის დაყენებული თითოეული პაკეტის.
ეს უზრუნველყოფს, რომ ყველას ერთნაირად დაეყენოს ბიბლიოთეკები.

vite.config.js

Vite-ის კონფიგურაციის ფაილი.
აქ წერია პროექტის build/გაშვების პარამეტრები, დამატებითი პლაგინები და სხვა სეტინგები.

index.html

ეს არის მთავარი HTML ფაილი, რომელსაც ბრაუზერი სატარებად იღებს.
შიგნით React აპლიკაცია თავსდება ერთი div-ში (root), და დანარჩენს React მართავს.
